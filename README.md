# OpenClaw 7 Days 实战课程

> 从零开始，掌握 AI Agent 开发与部署

🖥️ **在线课程网站**: https://openclaw-course-silk.vercel.app

---

## 课程介绍

本课程面向希望掌握 OpenClaw 的开发者，通过 8 节实战课程，层层递进地学习 OpenClaw 的各项核心功能。

**课程特色：**
- 🛠️ 每节都有动手实战
- 📚 真实的案例驱动
- 🔰 从基础到生产部署
- 📝 完整的命令示例

---

## 课程大纲

| 课次 | 主题 | 核心内容 |
|:---:|------|---------|
| **Lesson 1** | OpenClaw 基础与环境搭建 | 安装配置、架构概念、运行第一个 Agent |
| **Lesson 2** | Agent 创建与管理 | Agent 配置、多 Agent 架构、工具权限 |
| **Lesson 3** | 渠道集成配置 | 飞书、Telegram、微信渠道接入 |
| **Lesson 4** | Skills 技能系统 | 内置 Skills、自定义 Skill 开发 |
| **Lesson 5** | 记忆与上下文管理 | Memory 系统、混合搜索、上下文压缩 |
| **Lesson 6** | MCP 工具集成 | MCP 架构、自定义 MCP 服务器 |
| **Lesson 7** | 自定义工作流 | 工作流编排、并行任务、条件分支 |
| **Lesson 8** | 生产部署与运维 | Docker 部署、监控告警、高可用 |

---

Lesson 1 → Lesson 2 → Lesson 3 → Lesson 4 → Lesson 5 → Lesson 6 → Lesson 7 → Lesson 8
  ↓          ↓          ↓          ↓          ↓          ↓          ↓          ↓
